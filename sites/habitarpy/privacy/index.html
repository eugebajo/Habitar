Politica de privacidad

Version preliminar para la primera version de Habitar. Ultima actualizacion: 23 de julio de 2026.

Responsable

Habitar es una aplicacion de acompanamiento familiar para rutinas, habitos y autonomia de ninos y adolescentes.

Contacto: soporte@habitarpy.com

Datos que podemos tratar

Podemos tratar datos de cuenta, datos familiares, perfiles de ninos o adolescentes, rutinas, habitos, progreso, preferencias de notificacion y check-ins opcionales de bienestar.

Para que usamos los datos

Usamos los datos para crear y proteger el espacio familiar, permitir login, mostrar a cada perfil la experiencia que corresponde, gestionar rutinas y sincronizar informacion cuando la cuenta lo permita.

Datos de ninos y adolescentes

Los perfiles infantiles y adolescentes son administrados por una persona adulta responsable. Los ninos y adolescentes no crean sus propias rutinas ni habitos desde su modo personal.

Comparticion de datos

No vendemos datos personales. Podemos usar proveedores tecnicos como Supabase, Google Play y Apple App Store para operar, distribuir y proteger la app.

Seguridad

La informacion se transmite mediante conexiones seguras. La base de datos de produccion debe usar reglas de acceso por usuario y familia. La app incluye un paso de confirmacion para proteger el espacio adulto.

Retencion y eliminacion

La persona adulta responsable podra solicitar soporte para revisar, corregir o eliminar informacion de su cuenta familiar. El flujo automatico de eliminacion de cuenta debe completarse antes del lanzamiento publico amplio.
