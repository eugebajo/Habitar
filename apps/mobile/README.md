# Habitar Mobile

Aplicacion Flutter principal del monorepo Habitar.

## Ejecutar

```powershell
flutter pub get
flutter analyze
flutter test
flutter run
```

- Registro adulto.
- Perfil infantil/adolescente.
- Panel familiar.
- Habitos.
- Rutinas guiadas.
- Recordatorios simulados.
