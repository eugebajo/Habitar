# Wear OS

Modulo nativo planificado para relojes Wear OS.

## Alcance inicial

- Actividad actual.
- Siguiente paso.
- Temporizador.
- Completar.
- Posponer.
- Pedir ayuda.
- Pausar y reanudar.
- Tile de rutina.
- Vibracion.

## Transporte previsto

- Backend comun para sincronizacion persistente.
- Notificaciones.
- Wear OS Data Layer API para comunicacion local Android-reloj.

Este directorio es preparatorio. El modulo Android/Wear real se crea en una fase de integracion nativa.
