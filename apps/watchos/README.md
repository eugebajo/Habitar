# watchOS

Modulo nativo planificado para Apple Watch.

## Alcance inicial

- Ver actividad actual.
- Ver proximo paso.
- Temporizador.
- Completar paso.
- Pedir mas tiempo.
- Pedir ayuda.
- Pausar y reanudar.
- Haptica suave.
- Complicacion o widget cuando sea viable.

## Transporte previsto

- Backend comun para sincronizacion persistente.
- Notificaciones.
- WatchConnectivity para comunicacion local telefono-reloj.

Este directorio es preparatorio. El target Xcode real se crea en una fase de integracion nativa.
