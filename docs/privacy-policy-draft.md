# Politica de privacidad de Habitar

Documento preliminar para la primera version de Habitar. Debe revisarse antes de
publicar en Google Play o App Store.

## Responsable

Habitar es una aplicacion de acompanamiento familiar para rutinas, habitos y
autonomia de ninos y adolescentes.

Contacto previsto:

- Email: `soporte@habitarpy.com`
- Sitio: pendiente de dominio propio de Habitar

## Datos que podemos tratar

Habitar puede tratar:

- Datos de cuenta: email, nombre del adulto, identificadores de autenticacion.
- Datos familiares: nombre de familia, perfiles de ninos/adolescentes, edad y
  tipo de perfil.
- Actividad dentro de la app: rutinas creadas, pasos, habitos, progreso,
  preferencias de notificacion, progreso de cuentos y uso de espacios.
- Datos opcionales de bienestar: check-ins emocionales y solicitudes de apoyo.

## Para que usamos los datos

Usamos los datos para:

- Crear y proteger el espacio familiar.
- Permitir login y recuperacion de acceso.
- Mostrar a cada perfil solo la experiencia que corresponde.
- Gestionar rutinas, habitos y progreso.
- Sincronizar informacion entre dispositivos cuando la cuenta lo permita.
- Mejorar seguridad, soporte y estabilidad de la app.

## Datos de ninos y adolescentes

Habitar permite crear perfiles infantiles y adolescentes administrados por una
persona adulta responsable. Los ninos y adolescentes no crean sus propias
rutinas ni habitos desde su modo personal; ven lo que el adulto, cuidador o
profesional autorizado preparo para ellos.

## Comparticion de datos

No vendemos datos personales.

Podemos usar proveedores tecnicos para operar la app, por ejemplo:

- Supabase para autenticacion, base de datos y sincronizacion.
- Google Play y Apple App Store para distribucion y servicios de plataforma.

## Seguridad

La informacion se transmite mediante conexiones seguras. La base de datos de
produccion debe usar reglas de acceso por usuario/familia. La app incluye un
paso de confirmacion para proteger el espacio adulto.

## Retencion y eliminacion

La persona adulta responsable podra solicitar soporte para revisar, corregir o
eliminar informacion de su cuenta familiar. El flujo automatico de eliminacion
de cuenta debe completarse antes del lanzamiento publico amplio.

## Cambios

Esta politica puede actualizarse a medida que Habitar incorpore nuevas
funciones. La version publicada debe indicar fecha de ultima actualizacion.
